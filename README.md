# Global Superstore Sales Analysis

## Project Description
The **Global Superstore Sales Analysis** project is a comprehensive data analytics portfolio project designed to examine retail sales, profitability, shipping patterns, and customer demographics across international markets. By performing data cleaning, feature engineering, outlier detection, and exploratory data analysis (EDA), this project unveils actionable business insights to help improve operational efficiency, optimize pricing strategies, and maximize revenue.

## Business Problem
Global Superstore operates across multiple geographical regions and product categories. However, leadership faces several key operational and profitability challenges:
- Low profit margins or financial losses in specific product sub-categories and geographic regions.
- High shipping costs and delays across different shipping modes.
- Inefficiencies in customer segmentation and product discount strategies.
- Identifying high-value customers and high-volume sales drivers to optimize marketing spend.

## Project Objectives
1. **Data Cleaning & Preparation**: Handle missing values, ensure consistent data types, and normalize datasets for seamless analysis.
2. **Feature Engineering**: Derive insightful metrics such as shipping duration, profit margins, and time-based indicators.
3. **Outlier Detection & Treatment**: Identify extreme values in sales, profits, and shipping costs to prevent skewed insights.
4. **Exploratory Data Analysis (EDA)**: Discover key sales patterns, regional performance trends, top-selling categories, and profitability drivers.
5. **Dashboard & Visualization**: Design an interactive Power BI dashboard to enable dynamic business decision-making.

## Tools & Technologies Used
- **Python**: Primary programming language for data manipulation and statistical analysis.
- **Pandas**: Efficient data loading, data manipulation, cleaning, and transformation.
- **NumPy**: Numerical calculations and data transformations.
- **Matplotlib**: Static visualizations and statistical charting.
- **Power BI**: Interactive business intelligence dashboard creation (Coming Soon).
- **Google Colab**: Cloud-based Jupyter Notebook environment for analysis and execution.

## Project Workflow
- **Data Understanding**: Inspecting dataset dimensions, data types, column distributions, and initial statistics.
- **Data Cleaning**: Imputing or removing missing values, removing duplicate records, and standardizing date formats.
- **Feature Engineering**: Creating custom metrics such as shipping duration, profit margins, and date components.
- **Outlier Detection**: Utilizing statistical methods to identify and analyze extreme values in sales and profit metrics.
- **Exploratory Data Analysis (EDA)**: Aggregating data across regions, categories, and timeframes to extract trend lines, correlations, and business performance metrics.
- **Power BI Dashboard (Coming Soon)**: Developing interactive dashboards featuring KPIs, regional maps, and category breakdowns.
